# AI-Chatbot-Webapp
This is a small project to host an AI chatbot in a browser. 
This is achieved through an HTML script that hosts all the properties of the chatbot. 
Inspired largely by J.A.R.V.I.S from Iron Man, it has been one of my goals to make a personalized AI, 
and this gets me one step closer to achieving that. 
Please make sure to replace the "Put_API_key_here" phrase with your own Google Gemini API key. 
Make sure not to share the API key with anyone else, especially online ;)
The rest of the codespace is used to give the chatbot a personality and configure the user interface. 
More optimizations to come later.
